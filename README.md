# Portfolio Website

This is my personal portfolio website, created using **HTML**, **CSS**, and **JavaScript**. The website showcases my skills, projects, and contact information, featuring my own images as well as additional assets stored in the `ini_files` folder.

## Project Structure

- `index.html` — Main HTML file for the website.
- `styles.css` — CSS file for styling the website.
- `script.js` — JavaScript file for interactive functionality.
- `files/` — Folder containing my personal images used on the website., Folder containing additional assets and files used in the project.

## Features

- Responsive design for mobile and desktop devices.
- Interactive elements built with JavaScript.
- Clean and modern layout highlighting skills and projects.
- Contact form integration for easy communication.

## How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
